# マヂカル.fm デザインシステム「関西ポップ」

マヂカル.fm のWebサイトと自動生成OGPが共有するデザイン言語の定義。
**「お笑いと駄菓子屋のエネルギー。賑やかだが、規則は厳密に統一されている」** が合言葉。

- サイト実装: `tailwind.config.mjs`（トークン）+ `src/layouts/Layout.astro`（CSS変数・共通クラス）
- OGP実装: `scripts/generate-og.jsx`（`C` オブジェクトが本書のトークンと同期）

---

## 1. コンセプト

| 原則 | 意味 |
| --- | --- |
| 色面分割 | セクションごとに背景色をガラッと切り替え、スクロールで世界が変わる楽しさを作る |
| 太いアウトライン | すべての浮いている要素は 3px のインク色アウトラインで縁取る（OGPは 4〜6px） |
| ハードシャドウ | ぼかしのない「ずれた影」。印刷物・ステッカーの物理感 |
| 丸くて太い文字 | 見出しは Mochiy Pop One、本文は M PLUS Rounded 1c。角ばった書体は使わない |
| 賑やかさの制御 | フォント2種・角丸・アウトライン・影の規則を全要素で統一し、色数で遊ぶ |

## 2. カラー

CSS変数（RGB三つ組）で定義し、Tailwind から `bg-lilac` `text-ink` 等で参照する。

### ベース

| トークン | ライト | ダーク | 用途 |
| --- | --- | --- | --- |
| `paper` | `#FFF8EE` | `#16132A` | ページ背景 |
| `surface` | `#FFFFFF` | `#231F3E` | カード・ボタンの地 |
| `ink` | `#1D1A2E` | `#F4EFE4` | 本文テキスト |
| `muted` | `#6E6884` | `#A9A3C0` | 補助テキスト |
| `edge` | `#1D1A2E` | `#0B0920` | アウトライン・ハードシャドウ |

### アクセント（両モード共通で鮮やか）

| トークン | 値 | 主な役割 |
| --- | --- | --- |
| `lilac` | `#CFB3F5` | ヒーロー・一覧ヘッダーの色面 |
| `lime` | `#C9E94E` | エピソードセクションの色面・再生ボタン |
| `tangerine` | `#FF8A3D` | **プライマリアクション**（再生CTA）・リンク・ロゴの「.fm」 |
| `sky` | `#7FD4F5` | お便り・エピソード詳細ヘッダーの色面 |
| `candy` | `#FF6FA5` | グッズセクションの色面 |
| `sun` | `#FFD23F` | ステッカーバッジ・ホバー背景・選択色・ティッカー帯 |

### ルール

- **鮮やかな色面（lilac 等）の上のテキストは常に固定ネイビー `#1D1A2E`**。テーマで切り替えない（`text-[rgb(29,26,46)]`）
- 色面の上に置くカードは `surface`（テーマ追従）。カード内テキストは `ink`/`muted`
- 紫一色のグラデーションは禁止。グラデーション自体を使わない（ベタ塗り＋ドット柄のみ）
- セクションの色順はトップページで lilac → sun(帯) → lime → tangerine → sky → candy を維持する

## 3. タイポグラフィ

| トークン | フォント | 用途 |
| --- | --- | --- |
| `font-display` | Mochiy Pop One（400のみ） | 見出し・ロゴ・ステッカー・話数チップ |
| `font-sans` | M PLUS Rounded 1c（500/700/800） | 本文。既定ウェイトは 500 |
| `font-mono` | IBM Plex Mono（400/500） | 再生時間・速度表示などの数値 |

- 巨大見出し（ヒーロー）には `text-shadow: 4px 4px 0 rgb(255 255 255 / 0.55)` の白ハードシャドウ
- リンクのホバーは色変化ではなく **3px の下線（decoration-tangerine）** を基本にする

## 4. 形状・立体

| トークン | 値 | 用途 |
| --- | --- | --- |
| アウトライン | `border-3`（3px solid `edge`） | ボタン・カード・バッジ・画像 |
| 角丸 | チップ/ボタン: `rounded-full` ・ カード: `rounded-2xl`(16px) ・ 大カード: `rounded-3xl`(24px) | それ以外の角丸は使わない |
| `shadow-pop` | `6px 6px 0 edge` | 大きなカード |
| `shadow-pop-sm` | `4px 4px 0 edge` | ボタン・エピソードカード |
| `shadow-pop-xs` | `2px 2px 0 edge` | 小さなチップ |
| ドット柄 | `.dotted`（edge 12% の水玉、22px 間隔） | 色面セクションの下地 |
| 回転 | ステッカーは ±2〜6° 傾ける。本文・カードは傾けない（お便りカードのみ -1° の例外） |

## 5. モーション

Emil Kowalski の原則に従う。**UIモーションは300ms以下・ease-out・transform/opacityのみ**。

| パターン | 実装 | 値 |
| --- | --- | --- |
| 押しボタン | `.btn-pop` | hover: 浮く（-1px + 影6px）/ active: 沈む（+2px + 影1px + scale 0.98）150ms |
| イージング | `ease-out-quart` | `cubic-bezier(0.23, 1, 0.32, 1)` |
| ステッカー浮遊 | `.float-a` / `.float-b` | 4.5〜5.2s ease-in-out infinite。装飾のみ |
| ティッカー帯 | `.ticker` | 26s linear infinite |
| 再生中表示 | `.eq`（イコライザー3本） | 0.7〜1.0s、行とボタンの状態表示 |
| 入場 | `.rise .rise-1〜4` | 480ms、60ms間隔の stagger。ヒーローのみ |
| プレイヤー出現 | translateY スライドイン | 300ms ease-out-quart |

- `prefers-reduced-motion: reduce` で浮遊・ティッカー・イコライザーは停止、riseはフェードのみ
- ホバー効果は `@media (hover: hover) and (pointer: fine)` 内のみ（Tailwind `hoverOnlyWhenSupported` 有効）
- キーボード操作（Space/←→/M/Esc/Shift+<>）にアニメーションは付けない

## 7. OGP（自動生成画像）

`bun scripts/generate-og.jsx` で生成。1200×630px。

### レイアウト

- 背景: アクセント色ベタ + インクのドット柄。**話数 % 4 で lilac → lime → sky → candy をローテーション**
- 白カード（角丸36 / 6pxアウトライン / 14pxハードシャドウ）に、話数ステッカー・配信日・タイトル・ブランド行
- サイト全体用 `public/ogp.png` は `--site` で生成

### 日本語組版ルール（scripts/generate-og.jsx）

satori に改行を任せず、行の組み方を自前で決定する:

1. **文節分割**: BudouX でテキストを文節に分割し、文節単位でしか折り返さない
2. **幅で判定**: 「何文字か」ではなく文字種ごとの幅テーブル（全角1.0 / 半角英数0.6 / 詰め約物0.58 …）の合計で収まりを判定
3. **サイズは諦めながら決定**: タイトルは 76→68→60→54→48px の順に試し、行数と行バランスが合格した瞬間に確定
4. **孤立行の防止**: 最終行が 2文字以下 or 先頭行の40%未満なら、行幅を段階的に狭めて再配分
5. **助詞を道連れに省略**: 収まらない場合のみ「…」省略。直前が漢字・英数字なら助詞ごと削る
6. **約物のアキ詰め**: 開き括弧は左、閉じ括弧・句読点は右を 0.42em 詰める（YakuHanJP 相当）
7. **絵文字**: Twemoji SVG を `graphemeImages` で埋め込み（`node_modules/.cache/twemoji` にキャッシュ）

### 運用

```bash
bun scripts/generate-og.jsx --latest 5        # 新着分のみ（既存はスキップ）
bun scripts/generate-og.jsx --latest 5 --force # 上書き
bun scripts/generate-og.jsx --site             # サイト全体用 ogp.png
bun scripts/generate-og.jsx --force            # 全話再生成（デザイン変更時）
```

## 8. Do / Don't

| Do | Don't |
| --- | --- |
| アウトライン3px・影・角丸の規則を守った新要素を作る | 影にぼかし（blur）を使う |
| 色はトークンから選ぶ | 新しい色コードを直書きする |
| 押せる要素に `.btn-pop` を付ける | hoverで色だけ変わる「押せなさそうな」ボタン |
| 色面の上の見出しは固定ネイビー | 色面の上で `text-ink`（ダークモードで白くなり読めない） |
| ステッカーだけ傾ける | 本文やカードを傾ける |
| フォントは3種のトークンから使う | 新しいフォントの追加・Inter/Roboto等 |
